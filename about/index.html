About this app

This is a quickly made SvelteKit app that uses much of the styling from the default template since I can't make anything look pretty :). Svelte is great though, I highly recommend it.

This app is a simplified version of the Auto-Modroll Web App. This app allows you to manually input any amount of usernames into a raffle. A drum roll is played during the selection process.

The intention of this app is to randomly select users that have entered into the raffle for a chance of mod. The app can also be repurposed for just randomly drawing users for any reason.

If you want to check out the code itself, here's a link to the github repository.

Inspired by TheFinie's original modroll app.
